# Skill 2 - Hibernate CRUD Operations

## Overview
Retail inventory system using Hibernate ORM for CRUD on Product records.

## Database Setup
```sql
CREATE DATABASE inventory_db;
USE inventory_db;
```

## How to Run
1. Update `hibernate.cfg.xml` with your MySQL password
2. Right-click `App.java` → Run As → Java Application

## Expected DB Output
| id | name   | description   | price | quantity |
|----|--------|---------------|-------|----------|
| 1  | Laptop | Gaming Laptop | 72000 | 8        |
