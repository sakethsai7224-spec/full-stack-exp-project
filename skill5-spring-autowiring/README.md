# Skill 5 - Spring Autowiring with @Autowired

## Overview
Demonstrates Spring Autowiring by injecting a Course object into a Student using @Autowired annotation.

## Project Structure
```
src/main/java/com/klu/
├── App.java
├── Student.java
└── Course.java
src/main/resources/
└── applicationContext.xml
```

## How to Run
1. Right-click`App.java` → Run As → Java Application

## Expected Output
```
Student ID : 10
Student Name : Divya
Gender : Female
Course : Course [id=301, course=Spring Core, dateOfCompletion=25-Jan-2026]
```
